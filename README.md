# 🧩 一筆書きスライドパズル (One-Stroke Sliding Puzzle)

一筆書きのルート構築と、クラシックなスライドパズルを融合させた新感覚の脳トレパズルゲームです。
ブラウザ上ですぐに遊ぶことができます！

## 🎮 プレイはこちらから
👉 **[(https://sah1r0.github.io/one-stroke-puzzle/)]**

---

## ✨ 主な特徴

- **3つの難易度（盤面サイズ）に対応**
  - **3 × 3**（壁 1 個）：サクッと遊べる入門編
  - **4 × 4**（壁 2 個）：一気に難易度が上がるやりごたえ十分の標準編
  - **5 × 5**（壁 3 個）：超難関のマスター編
- **スマートな「詰み防止」アルゴリズム**
  - 自動生成される壁の位置によって「絶対に解けない盤面」にならないよう、裏側でクリア経路の存在を自動検証してからゲームを開始します。
- **記録への挑戦（スコア機能）**
  - クリア時に **「かかった時間（秒）」** と **「クリック回数」** を測定・表示。自己ベスト更新を目指せます！
- **クリア鑑賞モード**
  - クリア後のリザルトウィンドウを「×」で閉じると、開通した綺麗な一筆書きのラインをじっくり眺めることができます。

## 🕹️ ルール
1. 青い数字タイルをクリックして、グレーの空きスペースにスライドさせます。
2. 黒い固定壁をまたいでタイルを移動させることはできません。
3. **「1」から最後の数字まで、壁を挟まずに隣り合わせで順番通りに繋がること**（1-2-3...）ができれば、一筆書き開通でクリアです！
